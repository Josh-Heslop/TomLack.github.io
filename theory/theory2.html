Chapter 5 - Algorithms

Linear Search

A linear search is the simplest of all searches and the program will simply check every single item in the list. If the program gets to the end of the list and what it is searching for isn't found then it isn't in the list.

Binary Search

A binary search is a faster way of searching a list than the linear search method. It is done by finding the midpoint and testing to see if the value that the porgram is searching for is higher or lower than the midpoint if it isn't equal. The part of the list that the value isn't in is disregarded and the remaining part of the list is treated as a new list and the process begins again. This carries on until the midpoint equals the searched for value. This this method of sorting does however require that the list is ordered.
